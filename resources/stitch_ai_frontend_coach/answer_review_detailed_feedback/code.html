QUESTION

Explain the difference between useMemo and useCallback

YOUR ANSWER SENT

useMemo is for memoizing values so they don't get recalculated every render. useCallback is for memoizing functions so they maintain the same reference between renders. This helps prevent unnecessary re-renders of child components that depend on those props.

auto_awesome
BETTER ANSWER

Both hooks optimize performance via memoization. useMemo returns the result of an invoked function, ideal for expensive calculations. In contrast, useCallback returns the function itself, maintaining referential identity to avoid breaking React.memo in children.
